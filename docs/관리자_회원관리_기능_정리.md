# 관리자 회원 관리 기능 정리

## 구현된 기능 (관리 팝업)

| 기능 | 설명 |
|------|------|
| **블록** | 회원 차단/해제. 차단 시 로그인·플레이 제한 (서버 연동 시 적용) |
| **캐시 지급/차감** | 보유 캐시에 ± 적용. 입력 숫자만큼 지급(양수) 또는 차감(음수) |
| **티켓 지급/차감** | 보유 티켓에 ± 적용 |
| **리워드 포인트 지급/차감** | 리워드 포인트에 ± 적용 |

위 항목은 회원 목록에서 [관리] 클릭 시 열리는 팝업에서 처리합니다.

---

## 추후 검토 가능 항목

| 항목 | 설명 |
|------|------|
| **회원 상세** | 플레이 기록, 참여 이벤트, 로그인 이력 등 상세 페이지/탭 |
| **검색/필터 강화** | 가입일 범위, 보유 재화 범위, 블록 여부, 최근 활동일 등 |
| **일괄 처리** | 선택 회원에게 일괄 지급/차감 또는 블록 |
| **활동 정지** | 블록과 별도로 기간 제한 정지 (예: 7일간 로그인 불가) |
| **닉네임/프로필 변경** | 운영자가 닉네임 변경 또는 프로필 초기화 |
| **탈퇴 처리** | 회원 탈퇴 처리 및 개인정보 삭제 정책 반영 |

서버 연동 시 API 설계에 맞춰 단계적으로 추가하면 됩니다.
